# CIB Banking Dashboard

## Project Overview

This project focuses on analyzing banking-related data and presenting key insights through an interactive Power BI dashboard. The objective was to transform raw data into meaningful visualizations that support better business understanding and decision-making.

## Project Workflow

### 1. Data Collection

Collected and consolidated raw banking data from multiple sources to create a unified dataset for analysis.

### 2. Data Cleaning & Preparation

Used Python to clean, preprocess, and validate the data by:

* Handling missing and inconsistent values.
* Removing duplicates and correcting data quality issues.
* Formatting and transforming data for analysis.
* Preparing the dataset for reporting and visualization.

### 3. Data Analysis

Performed exploratory analysis to identify trends, patterns, and key performance indicators (KPIs) relevant to banking operations.

### 4. Dashboard Development

Designed and developed an interactive Power BI dashboard that:

* Presents key metrics and performance indicators.
* Provides clear and visually appealing insights.
* Enables users to explore data through filters and interactive visuals.
* Supports data-driven decision-making.

## Tools & Technologies

* Python (Pandas, NumPy)
* Power BI
* Excel
* SQL

## Outcome

Successfully transformed raw banking data into a clean, structured dataset and developed a professional Power BI dashboard that delivers actionable insights through effective data visualization.
